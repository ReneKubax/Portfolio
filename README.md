## This is my personal portfolio with Tailwind CSS , Next.js and Typescript.


Build locally:

**1. Clone Repo**

```bash
git clone https://github.com/ReneKubax/Portfolio
```

**2. Install Dependencies**

```bash
npm i
```


**3. Run App**

```bash
npm run dev
```
